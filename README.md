# 推特文案润色 Skill V3.3

> **喂我你的文字，我学会你的腔调；不喂也行，我从咱聊天里自己学；给我碎片想法，还你一篇像你自己写的推特。**

## 这是什么

一个 Coze Agent Skill，让 AI 用**你自己的文风**写推特。不是模板化润色，是真正学会你说话的方式。

## 核心能力

### 🎤 文风喂养与模仿
喂入你的文章/推文/口播稿/语音转文字，自动拆解文风 DNA（语气词/句式/节奏/用词/情绪/开头/结尾/论证方式），生成文风档案，模仿你的腔调创作。

### 🤖 自动模式
你日常聊天中随口表达观点、吐槽、分享想法时，自动识别为推文素材，主动转化为推特文案。不需要说"帮我写推特"。

### 💬 从聊天历史学习
自动从你和 Agent 的聊天记录中提取文风特征。最真实的文风不是刻意喂的，是日常聊天里自然流露的。

### 📊 X/Twitter CSV 导入
支持从 X 专业版「账号分析 → 内容」直接导出 CSV，批量导入历史推文学习。自己真实发过的内容，文风样本最纯。

### 🔄 自进化机制
- **记忆清零**：每次清空上次创作状态，只保留文风档案+爆款结构库+迭代日志
- **反馈记忆**：用户纠正按四类记录，频次≥3次上升为硬性规则
- **爆款结构库**：三级存储（🟡待验证→🟢已验证→🔴高频复用）
- **进化指标**：一次通过率、返工次数、交付速度、爆款命中率

## 使用方式

### 在 Coze 中安装

1. 在 Coze Agent 设置中，添加 Skill
2. 搜索 `twitter-voice-clone` 或 `推特文案润色`
3. 添加到你的 Agent

### 触发方式

**自动触发**（无需手动指令）：
- 分享观点/判断/吐槽/经历
- 表达反常识结论
- 讨论 AI/技术话题并有明确立场

**显式触发**：
- "写推特"、"润色推文"、"改一下这条推"
- "学我的文风"、"喂你几篇我写的"
- "太AI了"、"改口语一点"

## 文风学习优先级

主动喂样本 + 自动学习 > 仅自动学习 > 默认风格
